Визначення ключових термінів

HTML (англ. HyperText Markup Language — мова розмітки гіпертексту) — стандартизована мова розмітки документів для перегляду вебсторінок у браузері.

CSS (англ. Cascading Style Sheets, укр. Каскадні таблиці стилів) — це спеціальна мова стилю сторінок[en], що використовується для опису їхнього зовнішнього вигляду.

JavaScript (JS) — динамічна, об'єктно-орієнтована[5] прототипна мова програмування. Реалізація стандарту ECMAScript.

Навчимось створювати посилання на сайти.

В цьому уроці ми навчимося створювати адаптивну сітку на CSS. Також навчимось працювати із змінними та функцією calc.

Процес навчання

Bootstrap — це безкоштовний набір інструментів з відкритим кодом, призначений для створення вебсайтів та вебзастосунків, який містить шаблони CSS та HTML для типографіки, форм, кнопок, навігації та інших компонентів інтерфейсу, а також додаткові розширення JavaScript. Він спрощує розробку динамічних вебсайтів і вебзастосунків.
Tailwind CSS — це структура CSS з відкритим кодом. Головною особливістю цієї бібліотеки є те, що, на відміну від інших фреймворків CSS, таких як Bootstrap, вона не надає серії попередньо визначених класів для таких елементів, як кнопки чи таблиці. Замість цього він створює список «корисних» класів CSS, які можна використовувати для стилізації кожного елемента шляхом змішування та зіставлення.
CSS flexbox — це CSS3 веб модуль. Flex дозволяє автоматично організовувати відповідні елементи в контейнері залежно від розміру екрана (або пристрою) та дає змогу використовувати більш ефективний спосіб верстки, вирівнювання й розподілу вільного місця між елементами у контейнері, навіть коли їх розмір невідомий і/або динамічний.
CSS Grid — це техніка в каскадних таблицях стилів, що дозволяє веброзробникам створювати складні адаптивні макети вебдизайну більш легко і послідовно у різних вебпереглядачах. Є й інші методи макета вебсторінок, що використовувались раніше, включаючи tables, CSS Box model та CSS Flexbox.
JavaScript паттерни (паттерн, от англ. design pattern) — повторювана архітектурна конструкція у сфері проектування програмного забезпечення, що пропонує вирішення проблеми проектування в рамках деякого контексту, що часто виникає.
